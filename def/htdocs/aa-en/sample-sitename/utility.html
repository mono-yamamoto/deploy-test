CSS Utilities Sample

The Standard Template provides a set of CSS utility classes that can be used on any element in order to modify the text, element placement or adjust the padding and margin.

  • SPACE
  • TEXT
  • DISPLAY
  • BACKGROUND

SPACE

These are to adjust the padding and margin.

margin

.mt-lg-** (PC) To adjust margin-top per 5px.
** is a multiple of 5 (fom 0 to 100).
.mt-sm-** (SP*1)
.mb-lg-** (PC) To adjust margin-bottom per 5px.
** is a multiple of 5 (fom 0 to 100).
.mb-sm-** (SP)
.ml-lg-** (PC) To adjust margin-left per 5px.
** is a multiple of 5 (fom 0 to 100).
.ml-sm-** (SP)
.mr-lg-** (PC) To adjust margin-right per 5px.
** is a multiple of 5 (fom 0 to 100).
.mr-sm-** (SP)

*1 SP means "Smart Phone".

padding

.pt-lg-** (PC) To adjust padding-top per 5px.
** is a multiple of 5 (fom 0 to 100).
.pt-sm-** (SP)
.pb-lg-** (PC) To adjust padding-bottom per 5px.
** is a multiple of 5 (fom 0 to 100).
.pb-sm-** (SP)
.pl-lg-** (PC) To adjust padding-left per 5px.
** is a multiple of 5 (fom 0 to 100).
.pl-sm-** (SP)
.pr-lg-** (PC) To adjust padding-right per 5px.
** is a multiple of 5 (fom 0 to 100).
.pr-sm-** (SP)

TEXT

These are to modify the text.

.pc-nowrap Nowrap (PC)

font

* You can modify the font color only if it has meaning; caution, emphasis and so on.

.font-weight-bold Font-weight: bold
.font-weight-normal Font-weight: normal
.text-xxlarge Font-size: 4.0rem（40px）
.text-xlarge Font-size: 3.2rem（32px）
.text-large Font-size: 2.8rem（28px）
.text-medium Font-size: 1.8rem（18px）
.text-middle
.text-red Font-color: #e73134
.text-333333 Font-color: #333333
.text-ffffff Font-color: #ffffff
.text-005bc5 Font-color: #005bc5

text-align

.text-center Text-align: center
.text-lg-center Text-align: center (PC)
.text-sm-center Text-align: center (SP)
.text-left Text-align: left
.text-lg-left Text-align: left (PC)
.text-sm-left Text-align: left (SP)
.text-right Text-align: right
.text-lg-right Text-align: right (PC)
.text-sm-right Text-align: right (SP)

vertical-align

.align-top Vertical-align: top

DISPLAY

These are to modify the element display and placement.

display

.visible-lg To make an element visible over 720px and to invisible under 720px.
.hidden-sm
.hidden-lg To make an element visible under 720px and to invisible over 720px.
.visible-sm
.d-block display: block
.d-inline display: inline
.d-lg-inline display: inline (PC)

float

.clearfix clearfix
.float-left float: left
.float-right float: right
.pull-right
.float-none float: none
